# Maze Ai(미로 경로 탐색 플랫폼)
## What is Maze AI 
> React와 BFS알고리즘을 사용하여 제작한 미로를 입력받으면 미로의 길을 찾아 출력해주는 플랫폼입니다.
## 사용한 기술 
> + HTML
> + CSS
> + JavaScript
> + React
> + StyledComponent
## 구현한 기능 
> + 미로 입력받을수있는 모달창 띄우기
> + 미로 입력받기
> + 미로 초기화
> + 미로의 길 찾기
> + 미로의 길 보여주기
> + 미로의 시작 도착이 있는지 여부 확인
> + 미로의 길이 없는지 확인
